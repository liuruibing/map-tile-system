# 🗺️ Python 瓦片拼接与生成系统

## 项目概述

将 `/home/map` 素材库中的散乱 PNG 图片无缝拼接为大幅影像，并生成 Web 墨卡托投影 (EPSG:3857) 的 XYZ 瓦片 (z=18,19,20)。

## 技术规范

- **素材解析**: 文件名包含左上和右下经纬度 (如 `Lxxx_113.66x22.73_113.67x22.72.png`)
- **核心逻辑**: 散乱 PNG 无缝拼接为大幅影像
- **瓦片生成**: z=18, 19, 20 三个等级
- **投影系统**: Web 墨卡托 (EPSG:3857), XYZ 瓦片规范

## 目录结构

```
map-tile-system/
├── src/           # 源代码
├── output/        # 拼接结果
├── tiles/         # 生成的瓦片
└── docs/          # 文档
```
